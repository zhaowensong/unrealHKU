# PDF issue 4 prototype: three real height points per building

The prototype is generated by
`Scripts/SignalRayDemo/build_pdf_issue_4_three_height_points.py`.

- Buildings sampled: `8`
- Points per building: `3` (`top`, `middle`, `bottom`)
- Total real collision points: `24`
- Artificial/fallback points: `0`
- Component class for every point: `CesiumGltfPrimitiveComponent`

The top point is revalidated with a downward complex-mesh trace and an
upward-facing normal. Middle and bottom are selected from one continuous
facade chain. Adjacent samples are rejected when their XY positions or surface
normals jump, which prevents a scan from silently switching to a neighboring
building.

Measured building spans in the verified prototype range from `7650 cm` to
`27150 cm`. This is the actual height variation used by the visualization,
rather than a uniform or randomly assigned building height.

`Saved/SignalRayDemo/pdf_issue_4_three_height_points.json`
